页面和资源目录如下：

```text
.
├── index.html

├── architecture.html
├── data.html
├── training.html
├── inference.html
├── code-map.html
├── experiments.html

└── assets/

    └── eval_images/
```

## 内容来源

教程基于 MiniMind-V 当前开源项目代码、README，以及 MiniMind-V / SigLIP2 / ALLaVA-4V 等公开资料整理。
